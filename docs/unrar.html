The RAR Format

1 Introduction

This document will describe the RAR format, since it is currently an undocumented format. It seeks to serve a similar role as the ZIP App Note. This document is very much a work-in-progress and is far from complete.

NOTE 1: This documentation cannot be used to create RAR-compatible archive programs like WinRAR. This documentation is only for the purposes of writing decompression software (i.e. unrar) in various languages and was reverse-engineered from the UnRAR source located at this page.

NOTE 2: This documentation will initially focus on what I believe is Version 3 of the RAR format.
